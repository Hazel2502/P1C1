Para la primera clase practica de este ciclo estaremos viendo un poco de HTML, CSS y JS como lo muestra a continuacion:

  • Diferencias entre HTML,CSS y JS
    • HTML CSS JS
    HTML es un lenguaje de marcado para crear páginas web. Usa etiquetas para estructurar y mostrar contenido, como texto, imágenes y enlaces. CSS da estilo a páginas web, controlando colores, diseño y apariencia visual. Se aplica a elementos HTML para mejorar la presentación. JavaScript agrega interactividad a las páginas web, permitiendo respuestas dinámicas a acciones del usuario.
